# Never-gonna-give-you-up-spam
Phishing to spam open the default browser to the YouTube video of Never Gonna Give You Up.

How It Works

1. Phishing Email: User receives an email with a link to a fake "Verify Your Account" page.

2. Malicious Website: When the user clicks the link, they're redirected to a page that automatically downloads .bat file.

3.Spamming YouTube: The .bat file opens the default browser and loops to open the YouTube link repeatedly.
